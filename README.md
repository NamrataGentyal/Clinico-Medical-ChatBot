# Clinico-Medical-ChatBot

# How to run?
### STEPS:-

### 1. Clone the repository

```bash
Project repo: git clone https://github.com/NamrataGentyal/Clinico-Medical-ChatBot.git
```
### 2. Create and Activate a virtual enviornment
```bash
conda create -n clinico python=3.10 -y
```

```bash
conda activate clinico
```
### Note: If you get this error: "CondaError: Run 'conda init' before 'conda activate'" after ruuning the above command Open anaconda prompt and run a command "conda init bash" and then close git bash and open it again then run the "conda activate clinico" command


### 3. install all requirements
```bash
pip install -r requirements.txt
```
